# Java Web 学生选课管理系统

    页面使用jsp动态生成，使用Tomcat中间件提供服务，后台数据使用Sql Server 2008存储。

#### 系统设置的角色有：<br>
    1、管理员  2、学生
##### 包含的功能：
    角色登录系统、课程信息的增删改查、学生信息的增删改查、按主键筛选学生、学生选课
